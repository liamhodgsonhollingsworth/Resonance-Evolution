# Name suggestions — Apeiron

Apeiron is the current working name (Greek philosophical concept — Anaximander's boundless source from which all things emerge). Any session working on this project can suggest a better name by appending an entry to the "Suggestions" section below; the maintainer reviews suggestions at their convenience.

This file deviates from the static-page convention so any session can edit it without the maintainer-in-the-loop constraint that would otherwise gate the README's name reference.

## Suggestions

- **2026-05-16, dream-features-skeleton session.** Candidates aligned with the maintainer's "dreams, lucid dreaming, dreamlike visuals" framing. Listed in rough order of how directly they capture the dream register without losing the architectural connotation Apeiron provides.

  - **Oneiron** — derived from Greek *oneiros* (dream); preserves Greek-philosophy register paired with Alethea (truth); rhymes with Apeiron so a minimal one-letter shift carries the meaning shift. *Captures:* the dream-substrate the project is becoming. *Drops:* the "boundless source" connotation, though "the dream-substrate" arguably implies it.
  - **Oneiros** — the Greek personification of dream itself; more proper-noun-shaped than Oneiron. *Captures:* dream-personified, mythological-register. *Drops:* the architectural-substrate connotation; reads more like a single character than a worldbuilding engine.
  - **Reverie** — English, immediate; reads as a familiar word rather than a coined one. *Captures:* daydream-state, freeform exploration. *Drops:* the philosophical lineage; risks reading as a product name rather than a research project.
  - **Hypnagogia** — the state between waking and sleep, the moment where ordinary causality bends. *Captures:* the threshold-state where the engine's rules feel both alive and dreamlike. *Drops:* easy pronunciation; medical-jargon register.
  - **Limen / Liminal** — Latin for threshold; the engine as the in-between space where any topology is possible. *Captures:* the architectural commitment to non-Euclidean traversal as the substrate, not the exception. *Drops:* direct dream connotation.
  - **Morpheus** — the Greek god of dreams who shapes them. *Captures:* dream-shaping, generative. *Drops:* heavily overloaded by The Matrix; reads as character-name.
  - **Somnia** — Latin for dreams; clean two-syllable shape. *Captures:* dream-collective. *Drops:* commercial sleep-aid associations.
  - **Phantasma / Phantasm** — Greek for apparition/dream-image. *Captures:* dream-as-image, projection. *Drops:* horror-register associations.

  **Recommendation if one must be picked:** Oneiron — the single-letter shift from Apeiron preserves project lineage while signaling the dream-mode reorientation; pairs naturally with Alethea; survives the "what does this word mean" test (it scans as Greek-derived but doesn't require the reader to already know the term). The recommendation is the proposer's preference, not a binding choice — the maintainer's call stands.

## Decision history

Pairs with Alethea in the Greek-philosophy register; fits the "infinitely generalizable" architectural commitment; evokes "the boundless source."
